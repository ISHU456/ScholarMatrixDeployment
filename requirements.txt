# Project Requirements - LMS

## Technology Stack
- **Frontend**: React + Vite + Tailwind CSS + Framer Motion
- **Backend**: Node.js + Express + MongoDB + Socket.io
- **Database**: MongoDB (Local or Atlas)
- **AI Integration**: Google Generative AI (Gemini) + Groq SDK
- **Face Authentication**: Face-api.js (Browser-side)

## Development Prerequisites
- Node.js (v18.0 or higher)
- npm or yarn
- MongoDB Server

## Core Backend Dependencies (server/package.json)

## Scripts
- Run `npm install` in the root folder to install all dependencies.
- Run `npm run dev` in the root folder to start both frontend and backend.
- Use `npm run seed` to seed the database with initial data.
